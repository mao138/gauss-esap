Eliminación de Gauss Interactiva

Área de Matemáticas · Programa de Administración Pública

Recurso didáctico para apoyar el aprendizaje del método de eliminación de Gauss en la resolución de sistemas de ecuaciones lineales mediante operaciones elementales por filas.

Matriz inicial

Ingresa una matriz aumentada escribiendo cada fila en una línea diferente y usando comas para separar las columnas. Para números decimales usa punto decimal, por ejemplo 1.5 o -0.25. También puedes usar fracciones como 1/2, -3/4 o 5/2.

Ejemplo:
1,3,1
2,-1,9

Operaciones elementales por filas

2. Multiplicar una fila por un escalar

← · 1
Puedes usar enteros, decimales con punto o fracciones, por ejemplo: 2, -1.5, 1/2.

3. Reemplazar una fila por la suma de un múltiplo de otra

← 1 + ·
Ejemplo: R2 ← R2 + (1/2)·R1
